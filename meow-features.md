# 创意喵 2.0 功能设计

创意喵 AI 内容众包撮合平台，全产品功能设计，覆盖用户端与后台。

## 一、产品核心定位

创意喵 = 中小企业内容众包撮合 + 双向信用管控 + 效果可追溯。

主打：

- 低成本
- 高效率
- 强落地

目标：

- 让商家快速获取优质内容
- 让创作者轻松变现
- 让平台形成撮合与风控闭环
- 聚焦“优质内容采购”，为后期流量分发功能落地奠定基础

## 二、用户端功能设计

用户端覆盖 `APP + 小程序`，双端同步。

### 模块 1：商家端（需求方）

#### 1.1 入驻与个人中心

- 基础入驻：填写商家名称、Logo、联系人、联系电话，可选填公司地址、营业执照
- 轻量化入驻：大额任务需补全认证
- 账号管理：修改基础信息、更换 Logo、绑定收款账户、设置登录密码/手势密码
- 余额管理：
  - 充值：支持微信、支付宝、对公转账
  - 提现：未使用赏金原路退回
  - 账单明细：充值、赏金支出、退款、平台抽成
- 信用中心：查看信用分、信用记录、信用等级
- 消息通知：投稿审核提醒、任务到期提醒、申诉结果提醒、系统公告

#### 1.2 任务发布与管理

##### 任务创建

- 基础信息：
  - 任务标题
  - 任务分类：短视频、图文、AI 绘画等
  - 任务描述：富文本，支持换行、加粗、参考链接
- 素材上传：
  - 支持视频、图片、文档
  - 支持多文件上传
  - 单个文件不超过 500M
  - 支持重点需求标注
- 时间设置：
  - 投稿截止时间
- 奖励设置：
  - 基础参与奖：审核通过即得，可设置单条金额和参与上限
  - 排名奖励：一等奖、二等奖、三等奖、优秀奖，可分别配置数量和金额
- 资金托管：
  - 发布前必须预付全部赏金
  - 平台实时计算总预算
  - 预付成功后任务才可上线
- 任务设置：
  - 是否公开任务
  - 是否允许重复投稿
  - 是否开启 AI 查重

##### 任务管理

- 投稿列表：查看待审核、已通过、已驳回投稿，可按时间、审核状态、评分排序
- 作品操作：
  - 审核通过 / 驳回
  - 驳回需填写具体原因
  - 下载原素材
  - 收藏优质作品
  - 额外打赏
  - 评分（1-5 星）+ 文字评价
- 批量操作：
  - 批量审核
  - 批量下载
  - 批量评选获奖作品
  - 批量导出投稿数据
- 任务编辑：
  - 投稿截止前可修改任务要求和奖励金额
  - 若提高奖励需补充差额赏金
  - 截止后不可修改
- 任务终止：
  - 可申请终止任务
  - 平台审核后退回未发放赏金
  - 已产生的基础奖励不退

#### 1.3 内容效果查看

- 单任务数据大盘：
  - 总投稿数
  - 审核通过数
  - 总奖励支出
- 单作品数据：
  - 作品详情
  - 商家评分与评价
  - 奖励明细
  - 可手动标注作品使用情况
- 数据导出：支持导出 Excel
- 爆款标注：可手动标注优质爆款作品，用于后续优先邀请

#### 1.4 申诉与纠纷处理

- 申诉发起：针对抄袭、无关内容等恶意投稿上传证据
- 申诉查看：查看处理进度、裁决结果、裁决依据
- 纠纷反馈：联系客服，上传凭证，跟踪处理进度

### 模块 2：创作者端（供给方）

#### 2.1 注册与个人中心

- 注册登录：支持手机号、微信授权登录
- 绑定实名信息：用于提现与合规
- 账号管理：昵称、头像、简介、提现账户、隐私设置
- 收益管理：
  - 待解冻余额
  - 可提现余额
  - 收益明细：基础奖、排名奖、打赏
  - 提现：任务结束后解冻，T+1 到账
- 信用中心：查看信用分、信用记录、信用等级、违规原因、申诉入口
- 消息通知：投稿审核结果、奖励到账提醒、商家打赏提醒、任务邀请提醒、系统公告

#### 2.2 任务参与与投稿

- 任务浏览：
  - 按分类、奖励金额、截止时间、任务热度筛选
  - 查看任务详情：要求、参考素材、奖励规则、商家信用
- 任务报名：
  - 点击报名参与
  - 支持收藏任务
  - 报名后可查看自身投稿记录
- 投稿操作：
  - 上传视频、图片
  - 支持多文件
  - 单个文件不超过 500M
  - 可添加作品描述
  - 提交后进入待审核状态
  - 审核通过前可修改或删除投稿
- 投稿限制：
  - 信用分 `>= 80`：不限投稿数量
  - `60 <= 信用分 < 80`：每日最多 10 条
  - `40 <= 信用分 < 60`：每日最多 5 条
  - `信用分 < 40`：禁止投稿
  - 单个任务最多投稿 3-5 条，由商家自定义

#### 2.3 作品管理

- 查看所有投稿作品：待审核、已通过、已驳回
- 查看商家评分与评价
- 支持删除自己的投稿作品
- 审核通过后删除不影响已获得的基础奖励

#### 2.4 申诉与反馈

- 申诉发起：针对恶意驳回、未按时发放奖励、恶意评价等发起申诉并上传证据
- 申诉查看：查看进度、裁决结果、裁决依据，可补充二次证据
- 意见反馈：提交功能建议、规则建议、系统异常反馈并查看进度

## 三、总后台功能设计（平台官方端，PC 版）

### 模块 1：系统总览（数据看板）

- 平台总用户数：商家 + 创作者
- 活跃用户数
- 今日 / 昨日 / 本月任务数：发布中、进行中、已结束
- 总交易金额
- 平台总抽成
- 申诉处理率
- 违规率
- 数据图表：
  - 用户增长趋势图
  - 任务发布趋势图
  - 交易金额趋势图
  - 支持按日 / 周 / 月筛选
- 异常提醒：
  - 未处理申诉
  - 恶意违规用户
  - 资金异常
  - 系统故障

### 模块 2：用户管理

#### 2.1 商家管理

- 商家列表：入驻状态、信用分、注册时间、累计发布任务数、累计消费金额
- 支持按状态、信用分、时间筛选
- 商家操作：
  - 查看详情
  - 审核认证信息
  - 修改信用分
  - 限制发布权限
  - 封禁账号
  - 导出商家数据
- 商家投诉：处理创作者对商家的投诉并记录裁决结果

#### 2.2 创作者管理

- 创作者列表：注册状态、信用分、注册时间、累计投稿数、累计收益
- 支持按状态、信用分、收益筛选
- 创作者操作：
  - 查看详情
  - 审核实名信息
  - 修改信用分
  - 限制投稿权限
  - 封禁账号
  - 导出创作者数据
- 创作者投诉：处理商家对创作者的投诉并记录裁决结果

### 模块 3：任务管理

- 任务列表：发布状态、商家、任务类型、奖励预算、投稿数、进度
- 支持按状态、类型、时间筛选
- 任务操作：
  - 查看详情
  - 审核任务内容
  - 修改任务状态：上线、下架、终止
  - 删除违规任务
  - 导出任务数据
- 任务监控：
  - 实时监控投稿情况和审核情况
  - 识别恶意悬赏、垃圾投稿过多等异常任务

### 模块 4：资金管理

- 资金总览：
  - 平台总资金
  - 商家充值总额
  - 奖励发放总额
  - 平台抽成总额
  - 退款总额
- 商家资金：
  - 充值
  - 支出
  - 退款
  - 余额
  - 退款申请处理
  - 资金异常核实
- 创作者资金：
  - 收益
  - 提现
  - 冻结金额
  - 提现申请处理
  - 提现异常核实
- 抽成管理：
  - 设置抽成比例
  - 支持按任务类型、奖励类型自定义
  - 查看抽成明细
  - 导出资金账单
- 资金对账：
  - 每日 / 每月对账
  - 生成对账报表

### 模块 5：风控与申诉管理

- 申诉列表：查看全部申诉，按类型、状态筛选，查看证据
- 申诉处理：
  - 驳回申诉
  - 支持申诉
  - 部分支持
  - 填写裁决理由
  - 通知双方
  - 记录处理结果
- 违规管理：
  - 查看违规记录
  - 按类型、严重程度筛选
  - 处理方式：扣分、限制权限、封号
  - 记录违规处理结果
- AI 风控：
  - 设置视频 / 图片查重阈值
  - 设置违规内容识别规则
  - 实时监控投稿内容
  - 自动拦截违规投稿

### 模块 6：系统设置

- 系统参数设置：
  - 投稿上限
  - 信用分规则
  - 提现到账时间
  - 任务审核时限
  - 平台规则
  - 系统公告
- 权限管理：
  - 后台操作员账号
  - 分配操作权限
  - 修改密码
  - 注销账号
- 系统日志：
  - 后台操作记录
  - 用户操作记录
  - 系统异常日志
  - 支持按时间、操作人、操作类型筛选

### 模块 7：内容与版权管理

- 内容审核：人工审核 AI 未识别的疑似违规内容，删除违规内容并记录结果
- 版权管理：记录商家与创作者版权约定，处理版权纠纷，留存证据
- 内容归档：归档已结束任务投稿内容，支持按任务、商家、创作者查询

## 四、产品核心逻辑闭环

### 1. 商家端闭环

入驻 -> 充值 -> 发布任务（预付赏金） -> 审核投稿 -> 评选获奖 -> 查看内容效果 -> 任务结束 -> 资金结算（未用完赏金退回）

### 2. 创作者端闭环

注册 -> 浏览任务 -> 投稿 -> 审核通过 -> 任务结束 -> 收益解冻 -> 提现

### 3. 平台端闭环

监控任务 -> AI 风控 -> 处理申诉 -> 资金托管 -> 抽成 -> 数据统计 -> 系统维护

## 五、优化亮点

- 轻量化入驻：降低门槛，先入驻后完善认证，提升转化率
- 资金托管：解决商家跑路、创作者白干的信任问题，降低纠纷率
- AI 风控 + 人工审核：双重过滤违规内容与恶意投稿，降低人工成本
- 双向信用体系：约束商家与创作者行为，保障平台健康运营
- 自动结算：任务结束后自动解冻收益、退回未使用赏金，减少人工干预
